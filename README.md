# Interbot AI 🤖

An intelligent chatbot built with Python that can answer questions and hold conversations.

## 🚀 Features
- Natural language processing
- Interactive chat interface
- Context-aware responses

## 🛠️ Tech Stack
- Python
- NLTK
- Tkinter

## 🔧 How to Run
1. Clone: `git clone https://github.com/BunnyxBytes/interbot-ai`
2. Run: `python interbot.py`
